System Overview

This project bridges the physical and digital worlds by collecting real-time environmental metrics. Using high-precision sensors, we monitor Temperature, Humidity, Air and, Rain to create a comprehensive weather profile.

ESP32

Main microcontroller handling all processing and sensor communication.

DHT22

Measures ambient temperature and humidity.

MQ135

Detects air quality and harmful gases.

Detects rain intensity using analog output.

Libraries & Dependencies

These libraries enable sensor handling, display rendering, Wi-Fi features, and data logging for the Weather Station prototype.

1. ESP32 Built-in Libraries

WiFi.h

Handles ESP32 WiFi functions (AP mode, web server, connections).

Wire.h

Enables I2C communication (used by SSD1306 OLED display).

SPI.h

Used for SPI-based devices such as the Micro SD module.

SD.h

Reads and writes files to the SD card for data logging.

2. External Libraries (Installed)

DHT.h

DHT22 temperature & humidity sensor driver.
Source: Adafruit DHT Sensor Library

Adafruit_GFX.h

Core graphics library for drawing shapes, text, and rendering UI elements.
